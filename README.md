voici Tous les outils de développement installés. 

 1)- le moteur de web service AXIS 2:  
                                       
     Version: axis2-1.6.2
 
 2) le Serveur apache-Tomcat  
      
     Version:apache-tomcat-8.5.46

 3) le jar mysql
 
      Version: mysql-connector-java-8.0.17
 
 4) le JDK     
  
     Version : jdk1.8.0_181
 
 5) l'importation de la base de données  
   
        il faut importer la base de données TAHA_ELALAMI_DB.sql  
     
    le rapport TAHA_ELALAMI_RAPPORT.pdf contient le Guide complet d’installation.
